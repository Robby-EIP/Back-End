## <a name="Dependencies"></a> Dependencies
- ```sudo apt-get install python3```
- ```sudo apt-get install pip```
- ```sudo apt-get install pip3```
- ```pip install fastapi```
- ```pip install uvicorn```
- ```pip install platformio```

## <a name="HowToLaunch"></a> How to launch
- ```uvicorn main:app --reload```

or
- ```uvicorn main:app```
